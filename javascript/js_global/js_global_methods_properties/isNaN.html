JavaScript isNaN() details

In JavaScript NaN is short for "Not-a-Number".

The isNaN() method returns true if a value is NaN.

The isNaN() method converts the value to a number before testing it.

1. Sum Elements of a Array

Code

            const arr = [22, "11", "33", 44, "price: 55"]
            let sum = 0;
            for(let i =0; i < arr.length; i++){
                let element = arr[i];
                let convertedNumber = parseInt(element);
                if(!isNaN(convertedNumber)){
                    sum+=convertedNumber
                }
            }
            document.getElementById("sum").innerHTML = sum;
